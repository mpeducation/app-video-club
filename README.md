# Video Club

## Comenzando proyecto

```sh
touch server.js README.md && npm init -y && npm i express && npm i nodemon -D
```

## Dotenv

```sh
npm i dotenv -D
```

1. require('dotenv').config() 

2.a. const PORT = process.env.PORT 
2.b. const password = process.env.PASSWORD

3. Crear el archivo **.env**

```

PORT=8080
```
